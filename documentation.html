Frame Interpolation and Video Upscaling Project

By Roberto Tallarini, Lorenzo Gerosa & Gianluca Mariani

Processing Steps:

  • Frame Interpolation
    1. demo.py passes the video's path to the frame interpolation function.
    2. The video interpolation function increases the number of frames by first calculating optical flow between frame pairs both forward and backward, and then interpolating N new frames between each frame pair
    3. The original frames are discarded.
    4. The video is saved.
  • Video Upscaling
    1. demo.py calls the video upscaling function, passing it the path of the interpolated video. For each frame, the function invokes the bilinear upscaling function.
    2. Once a frame has been upscaled filters are applied, and the frame is saved to the new video until completion.
  • End of System

Legend:

Interpolation First Toggle:

Allows choosing whether to perform frame interpolation or upscaling first. By default, frame interpolation is done first. Computationally, it's more expensive to upscale first and then interpolate the frame rate.

Zoom Factor:

A decimal number that defines the upscaling factor.

Upscaling Iterations:

An integer indicating the number of upscaling iterations to perform. The resolution after upscaling is the initial resolution * (zoom factor upscale iterations).

Number of Frames to Interpolate:

An integer indicating the number of frames to interpolate between each frame pair (original frames are discarded). The final FPS after interpolation is the number of frames to interpolate * (initial frame count - 1).

Filters

During upscaling, image enhancements are applied, including blurring, Laplacian edge detection, and sharpening. All these filters are considered as a single filter in the graphical interface and applied in sequence according to an optimal algorithm. However, you can customize the parameters.

Select a Preset Filter:

Allows you to choose from a series of predefined filters.

Customize Filter:

Enables you to customize the parameters of the mentioned filters.

Show Edges:

Keeping this option off allows enabling or disabling the visualization of overlaid edge detection on the image.

Filter Parameters:

Kernel Size of First Applied Blurring:

This parameter represents the kernel size used to apply the blurring filter in the first stage of upscaling. The kernel size influences the degree of blurriness applied to the image. Larger values generate more intense blurriness.

Sigma X of First Applied Blurring:

Sigma X is a parameter of the blurring filter that affects the standard deviation of the Gaussian used for blurring. Higher Sigma X values generate more diffused blurriness, while lower values create more localized blurriness.

Central Value of Sharpening Kernel:

This parameter represents the central value of the kernel used to enhance image sharpness during upscaling. Increasing or decreasing the central value will affect the intensity of sharpening enhancement applied to the image.

Central Value of Laplacian Kernel:

The central value of the Laplacian kernel affects image sharpness and can be adjusted to achieve more or less intense edge detection effects. Increasing the central value can increase edge emphasis.

Binarization Threshold:

This parameter controls the threshold value used during the image binarization process. Binarization is a process that converts the image into binary form, where pixels above the threshold are considered edges. Adjusting the threshold value determines how many edges are included in the final image.

Kernel Size of Second Applied Blurring:

This parameter represents the kernel size used to apply the blurring filter in the second stage of upscaling. This stage can be used to apply additional blurriness to the image, if necessary. The kernel size influences the degree of blurriness applied.

Sigma X of Second Applied Blurring:

Sigma X is the parameter of the blurring filter used in the second stage of upscaling. Adjusting this value will influence the standard deviation of the Gaussian used for blurring in the second stage. Higher values generate more diffused blurriness, while lower values create more localized blurriness.
